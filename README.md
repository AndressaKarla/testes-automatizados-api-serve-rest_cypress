# :information_source: Introdução
Esse projeto "testes-automatizados-api-serve-rest_cypress" é executado em um ambiente de produção na ["API REST"](https://serverest.dev) do ["ServeRest"](https://github.com/ServeRest) que simula uma loja virtual, no navegador Chrome; no Sistema Operacional Windows 11, com o objetivo de praticar ainda mais testes automatizados de API REST em Cypress v13.15.0, cypress-plugin-api v2.11.2, Javascript e Commands.
